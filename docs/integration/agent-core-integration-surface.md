## 3 · RAG Retrieval API
**POST /v1/query**

Request
```json
{
  "query": "When is GA v3.0.0?",
  "top_k": 4
}
```

Response
```json
{
  "answer": "GA is targeted for 11 July 2025 [^1]",
  "citations": [
    { "id": "1433d0…", "excerpt": "GA v3.0.0 — 11 Jul 2025" }
  ]
}
```

Footnote numbers in `answer` map to the `citations` array.
